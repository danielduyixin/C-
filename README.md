# C语言 - - 大学生成绩管理系统
1. 此项目是用 C语言 实现的 < 大学生成绩管理系统 >
2. **项目实现的内容有：**
    1. 数据结构：
        1. 用C链表存储程序数据
        2. 与C链表相关的操作：
            1. 链表初始化
            2. 添加数据
            3. 查询数据
            4. 修改数据
            5. 删除数据
            6. 数据排序
            7. 链表长度动态计算
    2. 包含文件的读写操作
    3. 包含Windows创建目录的操作
    4. 包含C语言菜单设计和程序流程设计
    5. 包含C语言模拟密码输入的操作
    6. 包含输入密码后，删除文件所有内容，而不删除文件本身
    7. 可以随机重新生成相关数据
    8. 可以计算最高分，最低分和平均分
    9. 可以根据不同要求对程序数据进行排序
